Interactive Sorting
==========

Have fun with Sorting Algorithm:
* Ascending / Descending / Random / Input data source
* Start / Stop / Step precise control

Sorting algorithms include:
* Bubble Sort
* Insertion Sort
* Selection Sort
* Heap Sort
* Quick Sort
* Shell Sort
* Merge Sort (TopDown)
* Merge Sort (BottomUp)
